🏠 House Price Prediction using Linear Regression

This project demonstrates a simple Linear Regression model using:

NumPy

Matplotlib

Scikit-learn

The model predicts house prices (in Lakhs) based on area (in square feet) using a small 2D dataset.



📌 Project Overview

The script:

1. Creates a 2D dataset (Area vs Price)

2. Splits the data into training and testing sets

3. Trains a Linear Regression model

4. Evaluates the model using:

    - Mean Squared Error (MSE)

    - R² Score

5. Visualizes the regression line with the data points


📂 Dataset

The dataset is manually defined inside the script as a 2D array:

````[Area (sqft), Price (Lakhs)]
```

```[500, 30]
[1000, 38]
[1500, 45]
```

🛠 Technologies Used

1. Python

2. NumPy

3. Matplotlib

4. Scikit-learn


▶️ How to Run

Install Python (3.x recommended)

Install dependencies

Run the script:

```python main.py```


📊 Model Output

The script prints:

Intercept

Coefficient (slope)

Actual vs Predicted values

Mean Squared Error (MSE)

R² Score

It also displays a graph showing:

Scatter plot of original data

Best-fit regression line


📈 Example Output Visualization

X-axis → Area (sqft)

Y-axis → Price (Lakhs)

Blue dots → Data points

Line → Linear regression fit
